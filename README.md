# robofriends
robofriends is a small web app that allows you to search for robots made using react.

## Screenshots
[Demo video record](demo.gif)

## Usage
1. Clone this repo
2. Run ```npm install```
3. Run ```npm i --save redux-logger```
4. Run ```npm install --save redux-thunk```
5. Run ```npm start```
